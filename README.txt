Welcome to RoTool, this project is continuously being worked on by the developer. The purpose 
of this project, is to provide intresting add-ons to Roblox such as, a random group finder, a
random player finder, and a random avatar bundle finder. All you need to do to run this program is,
extract the folder, open the folder, open RoTool.html and from there, you are done, and can use the 
tools provided by the software. This project is solely kept updated by user support, this means
that this program will continue to stay free and not cost anything.
